Events

wsListen

Update content on WebSocket messages.


Options:

  • target (string, optional: false) specifies the target of the http request.
  • responseSelector (string, optional: true) a selector query of which to replace the innerHTML with the message. Uses document.querySelectorAll internally.

Methods:


    Example:

    { "$exampleEvent": { "type": "wsListen", "target": "ws://example.com/", "responseSelector": "#result" } }
